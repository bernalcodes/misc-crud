## creating a express backend

-	creating folder [server](./server/)
-	`npm init -y`
-	`npm install --save express mongoose dotenv nodemon`
-	create `server.js` file
-	add dev script to [package.json](./server/package.json)
	-	`"dev": "nodemon server.js",`
-	create `.env` file
